Thank you for downloading Easy FPS Editor. This is the introductory part for the manual and just informs you of the goals of this project.

Easy FPS Editor is a simple to use old-school First Person Shooter creation tool that allows you to create... well, old-school 3D FPS game. What does it mean? Imagine your own 3D shooter (or not as much a shooter and just a first person) game similar to Wolfenstein 3D, Doom or its clones with sprite-based graphics. Ever wanted to make one?

To keep it simple there are some restrictions, for example map size and number of floors you can set but in general this should be enough for a small and fun game. Especially considering there is unlimited number of maps per game.

It's main features are:

  • Tile based maps 64 by 64 tiles and 9 floors in height
  • Simple to use and understand editors including a map editor
  • Scripts for a wider control of game elements
  • MD3 models support when you don't want to use sprites anymore
  • Customizable (through shaders editing) shader posteffects
