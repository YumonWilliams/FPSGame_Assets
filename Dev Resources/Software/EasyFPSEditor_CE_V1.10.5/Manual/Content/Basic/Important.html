There are a few limitations on naming and other things. Nothing too serious though.

  • Editor and engine support only latin letters without whitespaces. The only exception is scripts code.
  • Do not load resources from the project folder itself!. The editor will copy them automatically where needed, renaming to proper names in process. Loading files directly from project folder might result in an error.
  • Even though nothing stops you, it's not a good idea to edit .dat files directly. Which should be obvious.
  • The editor doesn't clear unused files for you. You'll have to do that yourself before you release your game.
