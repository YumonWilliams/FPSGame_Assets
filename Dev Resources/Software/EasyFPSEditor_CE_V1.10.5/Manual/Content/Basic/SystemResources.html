The engine has some default resources that you will have to provide if you want them to be shown. They can be added in "Import System Resources" menu along with other system resources. The names here are selfexplanatory. Default effects sprites for enemies will be shown if they were not imported in enemies settings menu.

Menu images can optionally be set to a .vid file. If any menu image/video is missing, it will use a darkened version of the main menu image/video.
